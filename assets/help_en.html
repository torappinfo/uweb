Use multiple engines::
Type in main screen input or address bar, then click on any search engine.

Toolbar::
Swipe down at top or swipe at any bottom corner to show.
  • [⟳] reload/stop web page; long press for options.
  • <- : long press for auto page turning
  • -> : long press for app menu.
  • Settings: long press for options.
  • [□] tab management: long pressed to configure global javascripts.
  • Home screen: long press for new tab.
  • [☰] more tools; long press for global css.
More tools::
  • [PC] fake as desktop PC; long press for more UAs.
  • [JS] disable/enable javascript; long press for more javascripts.
  • [IMG] hide/show images; long press for more services.
  • [✰] add bookmark; long press for operations on selected texts.
  • [⌚] history, long press for user defined commands.
  • Search: long press for remote scripts.
  • Copy link, long press for menus for text selection.
  • Exit browser.
  • Go to the bottom.
  • Go to the top.
  • Speak: non-url address bar texts/selected texts/whole web page.
  • Share the url: share clipboard text when in home screen, otherwise share the page url.
  • Text reflow.
  • Download/Copy local files.
  • Switch profile.
Options::
Long press "settings" button to configure options.
  • adblock: block all subdomains.
    Need hosts file default.hosts.
  • Activate user script: when activated, the script will be run on similiar sites, until another script is activated or app exits.
  • Run user scripts for sites.
  • Resource sniffer: Show results when deselected. Use file types.sniff to sniff specified file types. Empty file to sniff all kinds.
