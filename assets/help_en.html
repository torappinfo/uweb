Use multiple engines::
Type in main screen input or address bar, then click on any search engine on the top part of home screen.

Toolbar::
Swipe down at top or swipe at any bottom corner to show.
  • [⟳] reload/stop web page; long press for spinner options.
  • <- : long press for auto page turning
  • -> : long press for app menu.
  • [⚙️]Settings: long press for options.
  • [□] tab showment; long press for global css, show "css/".
  • [🏠]Home screen: long press for new tab.
  • [☰] more tools; long pressed to configure global javascripts, show "js/".
More tools::
  • [🖥️]PC: fake as desktop PC; long press for more UAs, show default.uas.
  • [JS] disable/enable javascript; long press for more javascripts, show "bookmarklet/".
  • [IMG] hide/show images; long press for more services, show default.url.
  • [☆] add bookmark; long press for links menu, show default.link.
  • [🕔] history, long press for user defined commands, show default.cmds.
  • [🔍 Magnifier] search in the current page with button '>' or web search by ENTER; long press for remote scripts, show default.rjs.
  • [🔗]Copy link, long press for menus for text selection, show default.select.
  • [📤]Share the url: process clipboard text when in home screen, otherwise share the page url. Long press for input phrases, show default.input.
  • [X]Exit browser.
  • [▽]Go to the bottom.
  • [△]Go to the top.
  • [🔊]Speak: non-url address bar texts/selected texts/whole web page. Press again to stop.
  • [📄]Text reflow.
  • [☁↓]Download/Copy local files.
  • [👤]Switch profile.
Options::
Long press "settings" button to configure options.
  • adblock: block all subdomains.
    Need hosts file default.hosts.
  • Activate user script: when activated, the script will be run on similiar sites, until another script is activated or app exits.
  • Run user scripts for sites.
  • Resource sniffer: Show results when deselected. Use file types.sniff to sniff specified file types. Empty file to sniff all kinds.
Spinner options::
  • Background color
  • User agent
  • Address bar mode, show default.acmd
