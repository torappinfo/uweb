Information uweb browser collects

Uweb browser does not use any third-party SDK or library, but the browser do use system webview on your device to access the web. The system webview may leave trace of cookies and cache etc. These information should be disclosed by the privacy policy of the system webview. When uweb browser crashes, Crash report is generated on the device and keeps local only. All other information uweb browser save will need explicit user instructions like bookmarking, downloading, file saving, configuration etc.
